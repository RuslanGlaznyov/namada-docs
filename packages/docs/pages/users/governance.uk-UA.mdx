# Управління

Механізм управління Namada дає користувачам можливість динамічно оновлювати протокол.

Існує два різних механізми для створення пропозиції:

- [Пропозиції в ланцюжку](./governance/on-chain-governance.mdx): Пропозиція голосується і підраховується в ланцюжку. За бажанням може містити код пропозиції, який буде виконано, якщо пропозицію буде прийнято.
- [Пропозиції поза ланцюжком](./governance/off-chain-governance.mdx)
