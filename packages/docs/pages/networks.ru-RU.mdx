# Сети Namada

Эта страница предназначена для сетей Namada.

Существует два типа сетей:

- [Mainnets - основные сети](./networks/mainnets.mdx)
- [Testnets - тестовые сети](./networks/testnets.mdx)

Идентификаторы цепочек mainnet имеют префикс `main`, а идентификаторы цепочек testnet - префикс `test`.
